# eeun-link 운영 인수인계

## 서비스 개요
URL 단축 서비스. 긴 링크를 짧게 만들고, 클릭 통계와 만료일을 관리할 수 있음.

---

## 접속 주소
| 용도 | 주소 |
|------|------|
| 관리자 페이지 | https://eeun-link.vercel.app |
| 단축 URL 형태 | https://eeun-link.vercel.app/슬러그 |

---

## 사용 계정

| 서비스 | 용도 | 로그인 방법 |
|--------|------|------------|
| Vercel | 서버 (배포·운영) | vercel.com → GitHub 로그인 |
| Supabase | 데이터베이스 | supabase.com → GitHub 로그인 |
| GitHub | 코드 저장소 | github.com (eeuncontrol) |

- GitHub 아이디: `eeuncontrol`
- GitHub 레포: https://github.com/eeuncontrol/eeun-link

---

## 관리자 비밀번호 변경 방법
1. Vercel → eeun-link 프로젝트 → Settings → Environment Variables
2. `ADMIN_PASSWORD` 항목 → `...` → Edit → 새 비밀번호 입력 → Save
3. Deployments → 최근 배포 → `...` → Redeploy

---

## 환경변수 목록 (Vercel에 저장됨)
| Key | 설명 |
|-----|------|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase 프로젝트 URL |
| `NEXT_PUBLIC_SUPABASE_KEY` | Supabase API 키 |
| `ADMIN_PASSWORD` | 관리자 페이지 비밀번호 |

> API 키는 GitHub에 올라가지 않음 (.gitignore로 차단됨)

## 데이터베이스 구조 (Supabase)
Supabase → eeun-link 프로젝트 → Table Editor에서 확인 가능

### links 테이블 (링크 정보)
| 컬럼 | 설명 |
|------|------|
| id | 고유 ID |
| slug | 단축 코드 (예: rm3tw5) |
| name | 링크 이름 (예: 교보문고 신간) |
| original_url | 원본 URL |
| expires_at | 만료일 |
| is_active | 활성 여부 |
| created_at | 생성일 |

### clicks 테이블 (클릭 기록)
| 컬럼 | 설명 |
|------|------|
| id | 고유 ID |
| link_id | 연결된 링크 ID |
| clicked_at | 클릭 시간 |
| ip_address | 클릭한 IP |
| user_agent | 브라우저 정보 |

## 주요 기능
- **링크 생성**: 이름, 슬러그(직접 지정 or 자동), 만료일 설정 가능
- **클릭 통계**: 링크별 클릭 수, 일별 추이, 최근 클릭 기록
- **QR 코드**: 링크마다 QR 코드 자동 생성 및 다운로드
- **활성/비활성**: 링크를 삭제하지 않고 임시 중단 가능
- **만료일**: 지정한 날짜 이후 자동으로 접근 차단
